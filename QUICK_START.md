## 🏃‍♂️ How to Run

### Backend (Terminal 1)
```bash
# From project root
npm start
# Server runs on http://localhost:3000
```

### Frontend (Terminal 2)
```bash
# From project root
cd frontend
npm start
# Frontend runs on http://localhost:3001
```
